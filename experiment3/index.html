Experiment 3 - Generative Design with Circles

Imitate

During this first phase, I imitated an existing design (P_2_1_2_03) by switching from squares to circles. This required understanding the monochromatic color scheme and dynamic movement dependent on the cursor position from the original design, which would help me later on.

Integrate

During the integration stage, I used the map function to provide colour variations, which improved the background dynamics of the design. The smooth transition from squares to circles offered simplicity and versatility in selecting a single circle colour, however, I did experiment with different shapes earlier on. I contemplated later adding a feature to change the shapes but decided not to.

Innovate

During the invention phase, I added the ability to change the canvas dynamically, concealed the cursor in order to have a cleaner interface, and reversed colour elements to create variations that were more visually appealing. I mainly did that so that the circles and the background were more distinct. Also, the decision to make circles smaller and filled in, was influenced by comic book graphic gradients, because I always found them interesting and very appealing to look at.
